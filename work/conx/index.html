The Proccess

LogicalGroove team used a highly effective Agile methodology and well organized project managment. We were responsible for shaping the overall product look and interaction through wireframing, coming up with final design and choosing the backing technology.

Heavily based on Ruby on rails, is easy to use and highly interactive tool which users praise for it’s simplicity. Confident use of Ruby on Rails and various JS frameworks made the project engine and backend work perfect. Also we built a custom twitter bootstrap theme for rapid and responsive design.
